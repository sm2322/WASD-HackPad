# WASD HACKPAD

WASD HACKPAD is my hackpad that includes the basic wasd keys to be used for playing games! 

## Features:
- Simple Three Peice case
- 4 Keycaps
- custom QMK firmware!

## CAD Model:
Everything fits together using simple slide-in supports, it may not be perfectly study but it should make it useable!

It has 3 separate printed pieces, and includes supports for the pcb
